config

Templates can be customized my modifying the config

site_name: MkLorum
nav:
    - Home: index.md
    - Auto: my_module.md

plugins:
  - mkdocs_apidoc:
      function_template: |
        ### {{ name }}

        {{ signature }}

        {{ docstring }}

All functions will then be rendered via this jinja2 template instead of the provided default. The available arguments can be seen in models
